# 单位转换器 (Unit Converter)

一个优雅、直观的网页版单位转换工具，具有苹果风格的设计美学。

## 功能特点

- **多种单位转换类型**：
  - 温度（摄氏度、华氏度、开尔文）
  - 长度（米、厘米、千米、英寸、英尺、码、英里）
  - 重量（克、千克、磅、盎司、吨）
  - 容量（毫升、升、加仑、品脱、立方厘米）

- **直观的用户界面**：
  - 清晰的输入区域
  - 简单的单位选择
  - 实时转换结果
  - 适配所有设备尺寸

## 技术栈

- Next.js 14（React框架）
- TailwindCSS（样式）
- TypeScript（类型安全）

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

然后在浏览器中访问 http://localhost:3000

## 项目结构

- `/app` - Next.js 14 应用路由和页面
- `/components` - 可复用的React组件
- `/lib` - 工具函数和常量
- `/public` - 静态资源
